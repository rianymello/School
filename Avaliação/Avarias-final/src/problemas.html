Introdução Problemas comuns

Os problemas mais comuns

Problemas de energia:

  • O computador não liga.
  • O computador liga, mas não arranca.
  • O computador liga e arranca parcialmente.
  • O computador desliga aleatoriamente.
  • O computador liga, mas a fonte de alimentação faz ruídos estranhos.
  • O computador liga, mas a ventoinha da fonte de alimentação não funciona.

Problemas com o SETUP:

  • O computador está lento.
  • O computador apresenta erros de memória.
  • O computador aquece demais.
  • O computador apresenta artefatos gráficos.
  • O computador não reconhece os dispositivos USB.
  • O computador não emite som.

Problemas de vídeo:

  • Nada aparece no monitor.
  • Caracteres aleatórios aparecem na tela.
  • A imagem rola horizontal ou verticalmente.
  • As linhas superiores da imagem são mais largas que as inferiores.

Problemas com a motherboard:

  • Componentes não identificados.
  • A placa-mãe está danificada.

Problemas com a memória:

  • A cache está com defeito.
  • O upgrade da memória RAM parece não funcionar.
  • A memória RAM não é reconhecida.
  • A memória RAM está danificada.

Problemas com o processador:

  • O computador não arranca.
  • A cache L1 está com defeito.
  • O computador está superaquecendo.
  • O processador trava.

Problemas com a ROM:

  • O PC não consegue realizar o teste de inicialização.
  • Dispositivos não são detectados (HD, CD, DVD, FD).
  • A ROM está configurada corretamente.

Problemas com o disco rígido:

  • O disco rígido não funciona.
  • O disco não arranca e exibe a mensagem "C: drive failure insert boot disk".
  • O disco rígido funcionava bem, mas parou de funcionar após a troca da motherboard.
  • Não é possível inicializar com o disco rígido, mas ele funciona bem ao iniciar com um disco de sistema.
  • O sistema reconhece apenas 512MB de capacidade em um disco de grande capacidade.
  • O computador não consegue criar partições superiores a 2GB.
  • Erros ocasionais são encontrados durante a execução do scandisk.

Problemas com o teclado:

  • O teclado gera mensagens de erro.
  • O teclado não funciona.
  • O teclado não responde.

Problemas com outros periféricos:

  • A impressora não imprime.
  • A impressora imprime caracteres aleatórios.
  • O scanner não responde.
  • O mouse não se move.
  • O dispositivo não funciona.
